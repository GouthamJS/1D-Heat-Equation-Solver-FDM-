# 1D Heat Equation Solver (Finite Difference Method)

## Overview
This project solves the 1D heat equation using the Finite Difference Method (FDM), a numerical technique for solving partial differential equations.

## Equation
∂u/∂t = α ∂²u/∂x²

## Method
- Spatial discretization: Central difference
- Time discretization: Forward difference
- Stability condition applied

## Features
- Python implementation using NumPy
- Visualization using Matplotlib
- Animation of temperature evolution

## Skills Demonstrated
- Numerical Modeling (PDE)
- Scientific Computing
- Python Optimization
- Mathematical Problem Solving

## Output
Simulates heat diffusion across a rod over time.
